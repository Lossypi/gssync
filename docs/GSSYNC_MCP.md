# GSSync MCP — инструкции для Claude

Ты имеешь доступ к серверу **GSSync** — инструменту для синхронизации Google Sheets с локальными файлами (xlsx, json, csv). Используй его, когда пользователь просит что-то скачать из таблицы, загрузить данные обратно или проверить содержимое.

---

## Доступные инструменты

### Конфигурация

**`get_config`** — показать текущие настройки по умолчанию (URL таблицы, путь к файлу, формат).

**`set_config`** — сохранить настройки по умолчанию. Все параметры опциональны.
- `spreadsheet_url` — URL таблицы Google Sheets
- `file_path` — путь к локальному файлу
- `file_format` — формат: `xlsx` (по умолчанию), `json`, `csv`

---

### Просмотр листов

**`list_google_sheets`** — список листов в Google Sheets.
- `spreadsheet_url` *(обязательный)* — полный URL таблицы

**`list_local_sheets`** — список листов в локальном файле.
- `file_path` *(обязательный)* — путь к файлу
- `file_format` — формат файла (по умолчанию `xlsx`)

---

### Синхронизация

**`pull_sheet`** — скачать один лист из Google Sheets в локальный файл.
- `spreadsheet_url` *(обязательный)*
- `sheet_name` *(обязательный)* — название листа
- `file_path` *(обязательный)*
- `file_format` — по умолчанию `xlsx`

**`pull_all`** — скачать все листы из Google Sheets в локальный файл.
- `spreadsheet_url` *(обязательный)*
- `file_path` *(обязательный)*
- `file_format` — по умолчанию `xlsx`

**`push_sheet`** — загрузить один лист из локального файла в Google Sheets.
- `spreadsheet_url` *(обязательный)*
- `sheet_name` *(обязательный)*
- `file_path` *(обязательный)*
- `file_format` — по умолчанию `xlsx`

**`push_all`** — загрузить все листы из локального файла в Google Sheets.
- `spreadsheet_url` *(обязательный)*
- `file_path` *(обязательный)*
- `file_format` — по умолчанию `xlsx`

---

## Примеры использования

**Скачать всю таблицу:**
> "Скачай таблицу https://docs.google.com/spreadsheets/d/ABC в C:\data\report.xlsx"
→ вызов `pull_all`

**Загрузить один лист:**
> "Загрузи лист 'Бюджет' из C:\data\report.xlsx в таблицу"
→ вызов `push_sheet`

**Проверить, что в таблице:**
> "Какие листы есть в таблице https://...?"
→ вызов `list_google_sheets`

**Настроить дефолты один раз:**
> "Запомни: моя таблица https://..., файл C:\data\report.xlsx"
→ вызов `set_config`, затем можно использовать инструменты без явных параметров

---

## Важно

- Аутентификация через OAuth2. При первом запросе откроется браузер для входа в Google — это нормально.
- Токен сохраняется в `~/.gssync/token.json` и обновляется автоматически.
- Формулы сохраняются при синхронизации (не вычисленные значения).
