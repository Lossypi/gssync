# GSSync

Инструмент для синхронизации Google Sheets с локальными файлами (xlsx, json, csv), с MCP-сервером для работы прямо из чата Claude.

**Что умеет:**
- Скачивать и загружать целые листы Google Sheets
- Читать и обновлять отдельные строки без синхронизации всей таблицы
- Работать с форматами xlsx, json, csv
- Подключаться к Claude Desktop как MCP-инструмент

---

## Требования

- **Python 3.10 или новее** — [скачать на python.org](https://www.python.org/downloads/)
- **Claude Desktop** — [скачать на claude.ai](https://claude.ai/download)
- Аккаунт Google

## Установка

### Шаг 1. Скачать GSSync

```bash
git clone https://github.com/Lossypi/gssync.git
cd gssync
```

Или [скачай ZIP](https://github.com/Lossypi/gssync/archive/refs/heads/master.zip) и распакуй в любую папку.

### Шаг 2. Установить пакет

Открой терминал в папке с проектом и выполни:

```bash
pip install -e .
```

Это установит GSSync и все зависимости.

### Шаг 3. Получить credentials.json от Google

Чтобы GSSync мог обращаться к твоим таблицам, нужно создать ключ доступа в Google Cloud.

1. Открой [Google Cloud Console](https://console.cloud.google.com/)
2. Нажми **«Выбрать проект»** → **«Новый проект»**, дай ему любое название, нажми **«Создать»**
3. В меню слева: **API и сервисы** → **Библиотека**
4. Найди **«Google Sheets API»**, открой и нажми **«Включить»**
5. В меню слева: **API и сервисы** → **Учётные данные**
6. Нажми **«Создать учётные данные»** → **«Идентификатор клиента OAuth»**
7. Если появится запрос настроить экран согласия — нажми **«Настроить экран согласия»**:
   - Выбери **«Внешний»** → **«Создать»**
   - Заполни только обязательные поля (название приложения, email поддержки, email разработчика)
   - На остальных экранах просто нажимай **«Сохранить и продолжить»**
   - В конце — **«Вернуться на панель управления»**
8. Снова: **«Создать учётные данные»** → **«Идентификатор клиента OAuth»**
9. Тип приложения: **«Приложение для компьютеров»**
10. Нажми **«Создать»**
11. Нажми **«Скачать JSON»** (кнопка с иконкой загрузки)
12. Переименуй скачанный файл в `credentials.json`
13. Положи его сюда:
    - **Windows:** `C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\.gssync\credentials.json`
    - **Mac/Linux:** `~/.gssync/credentials.json`

> Папку `.gssync` создай вручную, если её нет.

### Шаг 4. Узнать путь к Python

Тебе понадобится полный путь к Python для настройки Claude Desktop.

**Windows** — в командной строке:
```
where python
```
Пример результата: `C:\Users\Vasya\AppData\Local\Programs\Python\Python312\python.exe`

**Mac/Linux** — в терминале:
```
which python3
```
Пример результата: `/usr/local/bin/python3`

### Шаг 5. Настроить Claude Desktop

Открой файл конфигурации Claude Desktop:

- **Windows:** `C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\AppData\Roaming\Claude\claude_desktop_config.json`
- **Mac:** `~/Library/Application Support/Claude/claude_desktop_config.json`

Добавь в раздел `"mcpServers"` следующее (замени пути на свои):

**Windows:**
```json
{
  "mcpServers": {
    "gssync": {
      "command": "C:\\Users\\Vasya\\AppData\\Local\\Programs\\Python\\Python312\\python.exe",
      "args": ["-m", "gssync.mcp_server"]
    }
  }
}
```

**Mac/Linux:**
```json
{
  "mcpServers": {
    "gssync": {
      "command": "/usr/local/bin/python3",
      "args": ["-m", "gssync.mcp_server"]
    }
  }
}
```

Если в файле уже есть другие серверы в `"mcpServers"` — просто добавь `"gssync"` рядом с ними через запятую.

### Шаг 6. Перезапустить Claude Desktop

Полностью закрой и снова открой Claude Desktop.

### Шаг 7. Первая авторизация Google

При первом использовании любого инструмента GSSync автоматически откроется браузер с запросом на вход в Google. Это нормально — нужно просто войти в аккаунт и разрешить доступ. Токен сохранится в `~/.gssync/token.json` и будет использоваться автоматически.

## Справочник команд

### Конфигурация

| Команда | Что делает |
|---------|-----------|
| `get_config` | Показать текущие настройки |
| `set_config` | Сохранить настройки по умолчанию |

`set_config` принимает любые из этих параметров:
- `spreadsheet_url` — URL таблицы Google Sheets
- `file_path` — путь к локальному файлу
- `file_format` — формат: `xlsx` (по умолчанию), `json`, `csv`

---

### Синхронизация листов целиком

| Команда | Что делает |
|---------|-----------|
| `list_google_sheets` | Список листов в таблице Google Sheets |
| `list_local_sheets` | Список листов в локальном файле |
| `pull_sheet` | Скачать один лист из Google Sheets в файл |
| `pull_all` | Скачать все листы из Google Sheets в файл |
| `push_sheet` | Загрузить один лист из файла в Google Sheets |
| `push_all` | Загрузить все листы из файла в Google Sheets |

---

### Построчная работа

Позволяет читать и изменять отдельные строки — без синхронизации всей таблицы.

**Фильтры** (нужно указать ровно один):

| Параметр | Пример | Смысл |
|----------|--------|-------|
| `row_numbers` | `"5"` · `"2,5,10"` · `"2:10"` | Строки по номеру (строка 1 = заголовок) |
| `filter_column` + `filter_value` | `"status"` · `"active"` | Все строки, где колонка = значение |
| `cell_range` | `"A2:D10"` | Диапазон в нотации A1 |

| Команда | Что делает |
|---------|-----------|
| `get_rows` | Прочитать строки из Google Sheets |
| `update_rows` | Обновить строки в Google Sheets |
| `pull_rows` | Скачать отфильтрованные строки в файл |
| `push_rows` | Загрузить отфильтрованные строки из файла в Google Sheets |

`update_rows` принимает `rows_data` — JSON-массив объектов вида `[{"колонка": "значение"}]`.
При использовании `filter_column` первый объект применяется ко всем найденным строкам.

## Примеры

**Скачать всю таблицу:**
> "Скачай таблицу https://docs.google.com/spreadsheets/d/… в C:\data\report.xlsx"

**Загрузить один лист обратно:**
> "Загрузи лист 'Бюджет' из C:\data\report.xlsx в ту же таблицу"

**Посмотреть конкретные строки:**
> "Покажи строки 3, 5 и 7 из листа 'Задачи'"

**Найти строки по значению:**
> "Покажи все строки где status = active в листе 'Задачи'"

**Обновить одну строку:**
> "Поставь status = done в строке 5 листа 'Задачи'"

**Обновить все совпадающие строки сразу:**
> "Смени статус с active на done у всех строк в листе 'Задачи'"

**Сохранить настройки один раз:**
> "Запомни: моя таблица https://…, файл C:\data\report.xlsx"

После этого можно не указывать URL и путь каждый раз.

## Форматы файлов

| Формат | Описание |
|--------|----------|
| `xlsx` | Excel-файл, один файл = все листы (по умолчанию) |
| `json` | JSON-файл, один файл = все листы |
| `csv` | Папка с отдельными .csv-файлами на каждый лист |

---

## Технические детали

- Авторизация: OAuth 2.0, токен хранится в `~/.gssync/token.json`
- Формулы сохраняются при синхронизации (не вычисленные значения)
- Настройки по умолчанию хранятся в `~/.gssync/config.json`
