# Установка GSSync MCP для Claude Desktop

Пошаговая инструкция для установки на любой компьютер. Следуй шагам по порядку.

---

## Что нам понадобится

- Компьютер с Windows или Mac
- Установленный [Claude Desktop](https://claude.ai/download)
- Аккаунт Google (тот, к чьим таблицам нужен доступ)
- ~20 минут

---

## Шаг 1. Установить Python

### Проверить, установлен ли уже

**Windows:** открой командную строку (нажми `Win + R`, введи `cmd`, нажми Enter) и напиши:
```
python --version
```
Если видишь что-то вроде `Python 3.12.0` — Python уже есть, переходи к Шагу 2.

**Mac:** открой Терминал (через Spotlight: `Cmd + Пробел`, введи «Терминал») и напиши:
```
python3 --version
```

### Если Python не установлен

1. Открой [python.org/downloads](https://www.python.org/downloads/)
2. Нажми большую кнопку **«Download Python 3.x.x»**
3. Запусти скачанный файл
4. **Важно:** на первом экране поставь галочку **«Add python.exe to PATH»** (она внизу)
5. Нажми **«Install Now»**

---

## Шаг 2. Скачать GSSync

1. Открой [github.com/Lossypi/gssync](https://github.com/Lossypi/gssync)
2. Нажми зелёную кнопку **«Code»** → **«Download ZIP»**
3. Распакуй ZIP в любое удобное место, например: `C:\gssync` (Windows) или `~/gssync` (Mac)

---

## Шаг 3. Установить GSSync

**Windows:**
1. Открой командную строку (`Win + R` → `cmd`)
2. Перейди в папку с проектом:
   ```
   cd C:\gssync
   ```
   *(или куда распаковал)*
3. Выполни:
   ```
   pip install -e .
   ```

**Mac:**
1. Открой Терминал
2. Перейди в папку:
   ```
   cd ~/gssync
   ```
3. Выполни:
   ```
   pip3 install -e .
   ```

Подожди — установятся все нужные библиотеки. В конце должно написать что-то вроде `Successfully installed gssync-0.1.0`.

---

## Шаг 4. Узнать путь к Python

Нам нужно знать, **где именно** установлен Python.

**Windows:** в командной строке:
```
where python
```
Скопируй первую строку из результата. Например:
```
C:\Users\Vasya\AppData\Local\Programs\Python\Python312\python.exe
```

**Mac:** в Терминале:
```
which python3
```
Скопируй результат. Например:
```
/usr/local/bin/python3
```

> Сохрани этот путь — он понадобится на Шаге 6.

---

## Шаг 5. Получить credentials.json от Google

Это самый длинный шаг — нужно создать ключ доступа в Google Cloud.

### 5.1. Создать проект в Google Cloud

1. Открой [console.cloud.google.com](https://console.cloud.google.com/)
2. Войди в тот Google-аккаунт, чьи таблицы будешь использовать
3. Вверху нажми на название текущего проекта (или «Выбрать проект»)
4. Нажми **«Создать проект»**
5. Введи любое название, например `gssync`, нажми **«Создать»**
6. Убедись, что в шапке выбран новый проект

### 5.2. Включить Google Sheets API

1. В левом меню: **API и сервисы** → **Библиотека**
2. В строке поиска введи `Google Sheets API`
3. Нажми на результат **«Google Sheets API»**
4. Нажми **«Включить»**

### 5.3. Настроить экран согласия OAuth

1. В левом меню: **API и сервисы** → **Экран согласия OAuth**
2. Выбери **«Внешний»** → **«Создать»**
3. Заполни обязательные поля:
   - **Название приложения:** `GSSync` (или любое)
   - **Адрес электронной почты для поддержки:** свой email
   - **Контактные данные разработчика:** свой email
4. Нажми **«Сохранить и продолжить»**
5. На следующих двух экранах («Области»  и «Тестовые пользователи») просто нажимай **«Сохранить и продолжить»**
6. В конце — **«Вернуться на панель управления»**

### 5.4. Создать учётные данные

1. В левом меню: **API и сервисы** → **Учётные данные**
2. Нажми **«Создать учётные данные»** → **«Идентификатор клиента OAuth 2.0»**
3. Тип приложения: **«Приложение для компьютеров»**
4. Название: любое, например `gssync-desktop`
5. Нажми **«Создать»**
6. Появится окно — нажми **«Скачать JSON»** (иконка загрузки)

### 5.5. Положить файл на место

1. Переименуй скачанный файл в `credentials.json`
2. Создай папку `.gssync`:
   - **Windows:** `C:\Users\ИМЯ_ПОЛЬЗОВАТЕЛЯ\.gssync\`
     *(имя пользователя — то, что написано в `C:\Users\`)*
   - **Mac/Linux:** `~/.gssync/`
3. Положи `credentials.json` в эту папку

> **Windows — как создать папку `.gssync`:**
> Открой Проводник, в адресной строке введи `%USERPROFILE%`, нажми Enter. Создай папку с именем `.gssync`.

## Шаг 6. Настроить Claude Desktop

### Найти файл конфигурации

**Windows:**
1. Нажми `Win + R`, введи `%APPDATA%\Claude`, нажми Enter
2. Открой файл `claude_desktop_config.json` любым текстовым редактором (блокнот подойдёт)

**Mac:**
1. В Finder нажми `Cmd + Shift + G`
2. Введи `~/Library/Application Support/Claude`
3. Открой `claude_desktop_config.json`

### Изменить файл

Найди в файле раздел `"mcpServers"`. Если его нет — создай файл с нуля.

Добавь блок `"gssync"` внутрь `"mcpServers"`:

**Windows** (замени путь к python.exe на свой из Шага 4, и путь к папке gssync):
```json
{
  "mcpServers": {
    "gssync": {
      "command": "C:\\Users\\Vasya\\AppData\\Local\\Programs\\Python\\Python312\\python.exe",
      "args": ["-m", "gssync.mcp_server"],
      "cwd": "C:\\gssync"
    }
  }
}
```

**Mac** (замени путь к python3 на свой из Шага 4, и путь к папке gssync):
```json
{
  "mcpServers": {
    "gssync": {
      "command": "/usr/local/bin/python3",
      "args": ["-m", "gssync.mcp_server"],
      "cwd": "/Users/vasya/gssync"
    }
  }
}
```

> **Если в файле уже есть другие серверы** — не удаляй их! Просто добавь `"gssync"` через запятую рядом с остальными.

> **Важно про обратные слеши в Windows:** в JSON каждый `\` нужно писать как `\\`. Путь `C:\Users\Vasya\...` в JSON пишется как `C:\\Users\\Vasya\\...`

### Сохрани файл

После редактирования сохрани файл (`Ctrl+S` или `Cmd+S`).

## Шаг 7. Перезапустить Claude Desktop

Полностью закрой Claude Desktop и открой снова. Просто крестик в углу не всегда помогает:

- **Windows:** найди иконку Claude в трее (правый нижний угол), нажми правой кнопкой → «Выйти»
- **Mac:** в меню вверху нажми Claude → «Quit Claude»

---

## Шаг 8. Первая авторизация Google

При первом запросе к таблицам автоматически откроется браузер с запросом входа в Google.

1. Войди в нужный Google-аккаунт
2. Возможно появится предупреждение **«Google не проверил это приложение»** — это нормально. Нажми **«Дополнительные настройки»** → **«Перейти на сайт gssync (небезопасно)»**
3. Разреши доступ к таблицам
4. Браузер покажет сообщение об успехе — можно его закрыть

Авторизация сохранится и больше не будет нужна (только если не выйти из аккаунта или не удалить токен).

или

## Что делать если не работает

**Claude не видит инструменты GSSync:**
- Проверь, что правильно указан путь к python в конфиге
- Проверь, что `cwd` указывает на папку, куда распаковал gssync
- Убедись, что выполнял `pip install -e .` из этой же папки
- Попробуй перезапустить Claude Desktop ещё раз

**Ошибка «No module named gssync»:**
- Открой командную строку, перейди в папку gssync и выполни `pip install -e .` ещё раз
- Убедись, что используешь тот же Python, что прописан в конфиге

**Браузер для авторизации не открывается:**
- Проверь, что `credentials.json` лежит в папке `~/.gssync/` (или `C:\Users\ИМЯ\.gssync\`)
- Убедись, что файл называется именно `credentials.json`, а не `credentials.json.json`

**Ошибка «The caller does not have permission»:**
- В Google Cloud Console проверь, что включён **Google Sheets API** (Шаг 5.2)
- Проверь, что `credentials.json` скачан из того же проекта, где включён API
